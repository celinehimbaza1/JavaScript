JS Array

Array is a Javascript special variable that can store more than one value

An array can hold many values under a single name, and you can access the values by referring to an index number.

Using an array literal is the esiest way to create Javascript array

Syntax:Const arrayName= [item1,item2,item3,...] and it is a common practice to declare an array with the const keyword

You can access an array element by referring to its index number

Arrays are Objects
    Arrays are a special type of objects. The typeof operator in JavaScript returns "object" for arrays.
    
    But, JavaScript arrays are best described as arrays.
    
    Arrays use numbers to access its "elements". In this example, person[0] 
    Objects use names to access its "members". In this example, person.firstName
